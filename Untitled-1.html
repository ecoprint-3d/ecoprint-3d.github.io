Архитектура системы "Эко Принт"

Многоуровневая архитектура автоматизированной системы сбора, переработки пластика и 3D-печати

Клиентский слой (Frontend)

Веб-интерфейс

  • Адаптивный дизайн для ПК, планшетов, смартфонов
  • Главная страница с описанием системы
  • Личный кабинет пользователя
  • Каталог 3D-моделей
  • Формы сдачи пластика и заказов
HTML5 CSS3 JavaScript React/Vue.js

Мобильный интерфейс

  • Упрощенная версия для мобильных устройств
  • Быстрый доступ к основным функциям
  • Push-уведомления
  • Камера для сканирования QR-кодов

Слой приложения (Backend)

Модуль аутентификации

  • Регистрация пользователей
  • Вход по логину/паролю
  • Восстановление пароля
  • Ролевая модель доступа

Модуль учета пластика

  • Внесение данных о сданном пластике
  • Привязка к пользователям и событиям
  • Автоматическое начисление бонусов
  • Учет типов пластика (PET, PP, ABS)

Модуль заказов и бонусов

  • Каталог 3D-моделей
  • Оформление заказов на печать
  • Расчет стоимости со скидками
  • Управление бонусной системой

Модуль администрирования

  • Панель управления администратора
  • Управление пользователями
  • Просмотр и управление заказами
  • Формирование отчетов
Python/Django Java/Spring Boot REST API JWT

Слой данных

База данных PostgreSQL

  • Хранение структурированных данных
  • Пользователи и роли
  • Учет пластика и заказов
  • Бонусные счета
  • Отчеты и аналитика

Файловое хранилище

  • 3D-модели в формате STL
  • Изображения изделий
  • Документация пользователей
  • Отчеты в формате PDF
